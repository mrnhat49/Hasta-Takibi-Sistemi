## ‚öôÔ∏è Setup the Hardware
1. Connect the DHT11 sensor to the Arduino following the schematic:
   - **VCC:** Connect to 5V Power
   - **GND:** Connect to Ground
   - **Data Pin:** Connect to any digital pin (e.g., D2)
2. Upload the provided Arduino sketch to your device using the Arduino IDE.
3. Ensure that your Arduino is connected to your computer.

## üìä Using the Application
1. Launch the installed application.
2. Choose the correct serial port for your Arduino from the application menu.
3. Click the "Start Monitoring" button.
4. View real-time data on temperature and humidity levels.
5. Set critical thresholds based on your requirements.
6. If conditions exceed these thresholds, the app will send alerts.
